</pre>

### Cafe Menu
<pre>
Design a cafe menu using CSS

<pre>

</pre>

<pre>

</pre>

### Rothko Painting
<pre>
Learn the Box Model - how each HTML element has its own spacing
Build a Rothko style painting using CSS and the Box Model
</pre>

<pre>

</pre>
